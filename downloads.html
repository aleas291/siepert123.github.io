Required ASM Libraries

These ASM (Java bytecode manipulation) libraries are required for MinecraftBorge to function properly. Download all three files.

ASM Core Library (v9.9)

Core ASM library required for bytecode operations.

Download asm-9.9.jar

ASM Tree Library (v9.9)

ASM tree manipulation library for advanced bytecode handling.

Download asm-tree-9.9.jar

ASM Commons Library (v9.9)

ASM commons library providing utility classes for bytecode manipulation.

Download asm-commons-9.9.jar

Installation Instructions

After downloading all files, follow these steps:

  1. Download minecraftborge.zip and all three ASM JAR files
  2. Add the ASM JAR files to your Minecraft jar using your preferred method (WinRAR, 7-Zip, or similar)
  3. Your installation is complete! You can now install mods by dragging them into the 'modifications' folder
  4. For mod development, download RetroMCP-Java and MinecraftBorge patch files and follow the Mod Development guide
